## ![icons8-tank-emoji-32](https://github.com/user-attachments/assets/1ddb6c3e-42c4-4ff6-9d7c-b0b33e33cbf3) Tanks  

**Developed and maintained by Ivan Hong**

A 2D tank battle game built with **Pixi.js**, where players control a tank, shoot projectiles, and fight against enemy tanks on a dynamic battlefield. The game includes multiple levels, obstacles, sound effects, and explosion animations, with full desktop and mobile support.
